SMART PET WEB

Com o objetivo de facilitar os médicos veterinários, criamos o sistema Smart pet web.

O sistema web terá o cadastro dos médicos veterinários e o cadastro dos animais. Com o sistema web, o veterinário terá um controle de sua agenda e poderá acompanhar os animais já cadastrados no sistema.
